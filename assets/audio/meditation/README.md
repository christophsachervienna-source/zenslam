# Meditation Audio Assets

This folder should contain the following audio files for the meditation timer feature:

## Required Files

1. **rain.mp3** - Rain ambient sound (loopable)
2. **forest.mp3** - Forest ambient sound with birds/nature (loopable)
3. **ocean.mp3** - Ocean waves ambient sound (loopable)

## Parent Folder (assets/audio/)

The parent folder should contain:
- **gong.mp3** - Meditation completion gong/bell sound

## Audio Requirements

- All ambient sounds should be high quality (192kbps+ MP3)
- Ambient sounds should be seamlessly loopable
- Gong sound should be ~2-5 seconds
- Recommended to keep file sizes reasonable for mobile (under 5MB each)

## Recommended Sources

- Freesound.org (CC licensed)
- Pixabay Audio (royalty-free)
- Zapsplat (with attribution)
